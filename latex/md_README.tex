{\bfseries{بصير (Baseer)}} is a flexible, C-\/based reverse engineering analysis tool built on a {\bfseries{Core + Extensions}} architecture. It can handle files from {\bfseries{any programming language}} and allows you to open files, extract byte blocks, and analyze them using modular, customizable extensions.

\DoxyHorRuler{0}
\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Architecture}{Architecture}}\label{md_README_autotoc_md2}
\hypertarget{md_README_autotoc_md3}{}\doxysubsection{\texorpdfstring{1. Core}{1. Core}}\label{md_README_autotoc_md3}
The Core handles the essential operations\+:


\begin{DoxyItemize}
\item {\bfseries{Open File}}\+: Read files in raw byte format.
\item {\bfseries{Extract Byte Blocks}}\+: load block of bytes into memory.
\item {\bfseries{Close File}}\+: Release resources when done.
\item {\bfseries{Handle Extensions}}\+: Load and manage various extensions.
\item {\bfseries{Manage Pipeline}}\+: Pass byte blocks through extensions sequentially or in parallel.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md4}{}\doxysubsection{\texorpdfstring{2. Extensions}{2. Extensions}}\label{md_README_autotoc_md4}
Extensions add advanced capabilities\+:


\begin{DoxyItemize}
\item {\bfseries{Fix Header}}\+: Repair file headers or order.
\item {\bfseries{Identify Block Type}}\+: Detect byte block type or metadata.
\item {\bfseries{Disassembler}}\+: Convert binaries to assembly instructions.
\item {\bfseries{Decompiler}}\+: Convert binaries to readable source code.
\item {\bfseries{Debugger}}\+: Dynamically monitor and analyze files.
\end{DoxyItemize}
